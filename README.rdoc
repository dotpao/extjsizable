= extjsizable

This gem insert the method to_ext_json into ActiveRecord and Array.
This method generates a Hash with the structure accepted by ExtJS.

{ total : N, data : [...] } in case of arrays.

{ success : true,  :data {...} in case of activerecord created/updated/destroyed data succesfully.
{ success : false, :errors {...} in case of activerecord created/updated/destroyed data failed.
